# Guia de Contribuição – ClimAtmos

Obrigado por se interessar em contribuir com o **ClimAtmos**, desenvolvido pela **Atmosmarine**.  
Este documento descreve, de forma resumida, como colaborar com o projeto.

---

## Formas de contribuir

Você pode contribuir de várias maneiras:

- **Reportando problemas (Issues)**  
  - Descreva claramente o problema.  
  - Inclua passos para reprodução, mensagens de erro, versão do Python e sistema operacional.  

- **Sugestões de melhorias e novas funcionalidades**  
  - Explique o contexto (caso de uso científico/operacional).  
  - Indique, se possível, referências técnicas ou artigos relevantes.

- **Pull Requests (PRs)**  
  - Correções de bugs.  
  - Melhorias de desempenho.  
  - Novos recursos bem justificados.  
  - Melhoria de documentação, exemplos e tutoriais.

---

## Fluxo básico para Pull Requests

1. Faça um **fork** do repositório e crie um **branch** a partir de `main` (ou `dev`, conforme indicado no README).
2. Implemente as mudanças, mantendo o estilo de código consistente (PEP 8 para Python, salvo indicação em contrário).
3. Sempre que possível, **adicione ou atualize testes**.
4. Execute a suíte de testes localmente antes de abrir o PR.
5. Abra o Pull Request descrevendo:
   - O problema que está sendo resolvido.  
   - A abordagem utilizada.  
   - Impactos esperados (científicos, de desempenho, compatibilidade, etc.).

## Licença das contribuições

O ClimAtmos é distribuído sob a [GNU General Public License v3.0 (GPLv3)](./LICENSE).

Ao enviar uma contribuição (código, documentação, dados de exemplo, scripts etc.), você declara que:

1. **É o autor** original da contribuição (ou está autorizado a contribuir em nome do detentor dos direitos);  
2. Sua contribuição **não viola direitos de terceiros**;  
3. Você concorda que sua contribuição seja **incorporada ao ClimAtmos** e **distribuída sob os termos da GPLv3**, em nome da **Atmosmarine** como titular do projeto.

Isso significa que:

- A Atmosmarine pode manter o ClimAtmos como um projeto coerente, com uma licença única (GPLv3);  
- Você mantém o crédito autoral nas mensagens de commit, histórico de Git e eventualmente na documentação;  
- Qualquer redistribuição do ClimAtmos (incluindo sua contribuição) continuará sujeita à GPLv3.

> **Nota:** Este documento não constitui aconselhamento jurídico.  
> Em caso de dúvidas legais específicas, consulte um profissional habilitado.
